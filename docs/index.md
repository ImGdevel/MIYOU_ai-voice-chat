# MIYOU Docs Index

이 디렉터리는 MIYOU 프로젝트 전용 문서의 루트 인덱스다.

## Sections

- `api/`
  프로젝트 API 계약과 요청/응답 규칙 문서를 모아두는 영역이다.
- `architecture/`
  프로젝트 아키텍처, 모듈 경계, 런타임 흐름 문서를 모아두는 영역이다.
- `convention/`
  프로젝트 전용 구현 규칙과 운영 규약 문서를 모아두는 영역이다.
- `domain-tech-spec/`
  도메인 동작, 상태 전이, 유스케이스 기술 문서를 모아두는 영역이다.
- `erd/`
  ERD, 테이블 정의, 관계 설계 문서를 모아두는 영역이다.
- `infrastructure/`
  배포, 클라우드, 런타임 인프라 문서를 모아두는 영역이다.
- `local-setup/`
  로컬 개발 환경 설치와 실행 문서를 모아두는 영역이다.
- `references/`
  외부 레퍼런스와 보조 자료를 모아두는 영역이다.
- `security/`
  인증, 인가, 시크릿, 보안 점검 문서를 모아두는 영역이다.
- `stack-selection/`
  스택 선택과 기술 트레이드오프 기록을 모아두는 영역이다.

## Existing Extra Sections

- `deployment/`
  기존 프로젝트에 이미 존재하던 추가 문서 디렉터리다.
- `guides/`
  기존 프로젝트에 이미 존재하던 추가 문서 디렉터리다.
- `spec/`
  기존 프로젝트에 이미 존재하던 추가 문서 디렉터리다.

## Rule

- 새 문서를 추가하면 가장 가까운 `index.md`를 같은 변경에서 갱신한다.
- 새 문서 디렉터리를 추가하면 이 루트 인덱스도 같이 갱신한다.
- `plan/`과 `troubleshooting/`는 프로젝트 루트에서 별도 관리한다.
